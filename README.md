# nidaqwrapper

Unified NI-DAQmx Python Wrapper.

A standalone Python package that provides a clean, high-level interface to
NI-DAQmx hardware, consolidating functionality from multiple projects into a
single maintainable codebase.

## Installation

```bash
pip install nidaqwrapper
```

With optional settings file support (pandas/openpyxl):

```bash
pip install nidaqwrapper[settings]
```
